## Overview

Enterprise AI Assistant is a multi-agent AI platform built using Streamlit and Gemini AI.

The system provides:

* AI Assistant Chat
* RAG-based PDF Question Answering
* HR Analytics Dashboard
* Employee Insights
* FAISS Vector Search
* Conversation Memory

## Features

✅ Multi-Agent Architecture
